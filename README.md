# 简介
一款为 HaaS / Falcon 平台打造的轻量级课程表小程序，支持课程持久化存储和周视图展示。

# 功能
- **课程管理**：添加、查看、删除课程
- **周视图展示**：支持周一至周日切换
- **两种时间模式**
  - 节次模式：内置 1~10 节
  - 自定义时间：滚轮选择起止时间
- **课程地点**：支持为课程设置上课地点，卡片右侧显示
- **数据持久化**：使用 SQLite 存储课程数据
- **关于页面**：展示应用版本、作者和框架信息

# 技术栈
- Falcon / HaaS UI
- Vue 2
- SQLite（通过 native 模块）
